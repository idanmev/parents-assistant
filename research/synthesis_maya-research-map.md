## 2. Routines & Transitions

**Key finding:** Transitions demand inhibition + mental set-shifting + working memory + emotion regulation + initiation — simultaneously. ADHD impairs all of these. Visual schedules reduce problem behavior and improve on-task behavior.

**The 3-phase transition failure model:**
1. Push original demand / escalate → **increases** dysregulation (AVOID)
2. Co-regulate first, reduce demands → stabilizes nervous system
3. Hours later, calm → upgrade the plan, rehearse

**What this explains in Maya's journals:**

| Journal observation | Research mechanism |
|---|---|
| Morning "eruptions" (school drop-off) | Home→school = highest-load transition of day |
| Dinner-to-bath resistance (multiple entries) | Activity switch demands all 5 executive functions at once |
| Markers obsession → can't stop drawing | Hyperfocus: harder to shift state than neurotypical kids |
| Timer worked / secret sign worked (Idan) | External tools offload working memory demand |
| "Let's hide from Mom to go shower" worked | Playful reframing reduces perceived transition threat |

**Natascha's response pattern (matches research):**
- "Timer + 5 minutes + secret sign" = visual + concrete transition structure
- Morning formula: Empathy → calm limit → one small step at a time
- Never re-escalate during meltdown → always co-regulate first

## 3. Emotional Dysregulation & RSD

**Key finding:** Emotional dysregulation is now considered a **core feature** of ADHD, not incidental. At age 6–7, RSD-type sensitivity shows as: disproportionate distress after mild correction, all-or-nothing thinking, rapid withdrawal/avoidance, clowning to pre-empt rejection.

**The evidence-based response sequence:**
1. Proactive setup (visual schedules, clear expectations)
2. During dysregulation → **connection before correction**, calm, minimal language
3. After partial calming → one clear command + immediate labeled praise
4. When fully regulated → brief collaborative review of what happened

**What this explains in Maya's journals:**

| Journal observation | Research mechanism |
|---|---|
| "I don't want to go to school" (friend went home) | Peer rejection sensitivity → school feels unsafe |
| Café scene (wanted to leave when classmate arrived) | Emotional flooding from ADHD sensory/social overload |
| Ice cream → can't choose → escalates | Exhausted brain + decision demand = impossible task |
| Testing limits through body (dinner) | Body-level boundary testing = regulation-seeking behavior |
| "Mommy, I already did my homework, look!" | Intense need for parental validation/approval |
| Never likes sharing food → now shares | Growing trust/security = reduced scarcity mindset |

**Natascha's framework (maps directly to BPT evidence):**
- "Empathy + calm limit + small steps = stability" → matches PMT protocol exactly
- "Connection before correction" → labeled explicitly in research as core principle
- "What you say and what happens must be predictable" → consistent contingencies
- Reinforcing micro-successes immediately → high-ratio labeled praise
